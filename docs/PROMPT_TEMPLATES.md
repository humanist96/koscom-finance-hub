## 1. UI 수정 프롬프트

### 1.1 텍스트/라벨 변경

```
현재 프로젝트의 [파일경로]에서 "[현재 텍스트]"를 "[변경할 텍스트]"로 수정해줘.
다른 부분은 건드리지 말고 해당 텍스트만 변경해.
```

**사용 예시**:
```
현재 프로젝트의 src/components/layout/header.tsx에서
"Securities Intelligence Hub"를 "금융동향파악"으로 수정해줘.
다른 부분은 건드리지 말고 해당 텍스트만 변경해.
```

### 1.2 버튼 텍스트 변경

```
[페이지명] 페이지에서 "[현재 버튼 텍스트]" 버튼의 텍스트를
"[새 버튼 텍스트]"로 변경해줘.
버튼의 기능이나 스타일은 그대로 유지해.
```

### 1.3 색상/스타일 변경

```
[컴포넌트명/파일명]의 [요소]의 색상을 [현재색]에서 [변경색]으로 바꿔줘.
Tailwind CSS 클래스를 사용해서 수정해줘.

예시 색상:
- 파란색: bg-blue-500, text-blue-600
- 녹색: bg-green-500, text-green-600
- 빨간색: bg-red-500, text-red-600
- 회색: bg-gray-500, text-gray-600
```

**사용 예시**:
```
대시보드 헤더 컴포넌트의 배경색을 bg-white에서 bg-slate-50으로 바꿔줘.
Tailwind CSS 클래스를 사용해서 수정해줘.
```

```

데이터는 [데이터필드]에서 가져오면 돼.

```

### 1.5 테이블 컬럼 순서 변경

```
[페이지명]의 테이블 컬럼 순서를 다음과 같이 변경해줘:

2. [두번째 컬럼]
3. [세번째 컬럼]
...
```

### 1.6 요소 숨기기/보이기

